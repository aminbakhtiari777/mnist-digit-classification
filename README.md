# MNIST Digit Classification

This is a beginner deep learning project for handwritten digit recognition using the MNIST dataset.

The project was built with TensorFlow and Keras and trained a neural network to classify digits from 0 to 9.

## What I practiced

- Loading image datasets
- Image visualization
- Data normalization
- Neural Networks
- Dense layers
- Activation functions
- Model training
- Model evaluation

## Technologies

- Python
- TensorFlow
- Keras
- Matplotlib

## Model Architecture

- Flatten Layer
- Dense Layer with ReLU
- Output Layer with Softmax

## Dataset

MNIST handwritten digits dataset.

## Result

The model was trained and evaluated on test images to classify handwritten numbers.
